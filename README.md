# Trail-Management-System
A cli app used to manage information about trails situated in the Zagorochoria region in Ioannina,Greece. Developed as part of a university course.

# Files info
a.txt and b.txt are used as example info the user can provide to the system in its initial use

After providing the input, the user can then fetch info about trails such as:
- All info about trails such as starting point, ending point, distance, maximum altitude and minimum altitude
- Which trails are mostly downhill and which are mostly uphill
- Which trails are the longest and which are the shortest trails
- etc.
